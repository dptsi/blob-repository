ITS Blob Repository (DEV)

Library for connecting to blob repo

Requirements

  1. PHP 5.4 ++
  2. guzzle

example is in test-upload.php

Instalation

add composer.json
    "myits/blob-repository": "dev-master"

after require section add this
    "repositories": [
        {
            "type": "vcs",
            "url": "https://git.its.ac.id/umar/blob-repository.git"
        }
    ]
